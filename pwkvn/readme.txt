This provides the working documents preparing the
digitization of the Nachtrage and Verbesserungen of PWK dictionary.

orig contains various 'editions' of the digitization in cp1252 format
with retention of the coding conventions adopted by Thomas Malten in
the typing of the dictionary text

step0 contains the many steps used to alter the original form of the
digitization.

install Discusses the changes required to various repositories
for a new pwkvn 'dictionary' in the Cologne sanskrit-lexicon.

step1: various editing
step1/althws  : revise the althws markup
